# CIDR / Subnet Calculator 🌐🔢

A professional-grade **IPv4 CIDR and Subnet Calculator** built with **Flutter**. This utility is designed for network engineers and IT professionals who need to perform rapid and accurate network address calculations on the go.

## 🌟 Key Features
- **Instant CIDR Calculation:** Calculate network addresses, broadcast addresses, and host ranges instantly.
- **Visual Subnetting:** Easily determine subnet masks and the number of available hosts.
- **Global Accessibility:** Fully localized in **50+ languages**, supporting IT professionals worldwide.
- **Developer-Friendly:** Open-source core logic for network bitwise operations.

## 💡 Best For...
- **Network Engineers:** Quick on-site network configuration and troubleshooting.
- **IT Students:** Learning and practicing IP addressing and subnetting.
- **SysAdmins:** Managing IP address allocations across different segments.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Efficient bitwise operations for IP address manipulation and mask generation.

*Note: This repository contains the core logic and source code. Assets (images, fonts, and localization files) are excluded to protect intellectual property.*
